# e-dialect 个人贡献者许可协议 (ICLA)

**致：** e-dialect 社区（以下简称“项目管理方”）

**说明：**
本协议是贡献者与管理方之间的法律协议。通过向 e-dialect 社区提交代码、文档或其他作品（以下简称“贡献”），贡献者同意以下条款，以保障项目的可持续发展和管理方对代码的灵活处置权。

### 1. 定义
“**贡献**”是指您提交给 e-dialect 项目的任何原创作品，包括但不限于代码、文档、设计、翻译等。
““**许可**”是指本项目采用的开源协议（目前为 GNU Affero General Public License v3.0 或后续版本）。

### 2. 版权许可授权
您特此授予项目管理方一份永久的、全球性的、免费的、非独占的、不可撤销的版权许可，包括以下权利：
1. 在现有或未来任何媒介上复制、修改、展示、分发您的贡献；
2. 制作衍生作品；
3. **再许可**：有权将您的贡献以“许可”或其他任何条款（包括但不限于商业闭源协议）进行再许可。

### 3. 专利许可授权
如果您的贡献中包含您拥有的专利权利要求，您特此授予项目管理方一份永久的、全球性的、免费的、非独占的、不可撤销的专利许可，允许项目管理方制造、使用、销售、许诺销售、导入您的贡献及其衍生作品。

### 4. 保证与陈述
您声明并保证：
1. 您是贡献的唯一作者，或者您有权代表第三方提交贡献；
2. 您的贡献是原创作品，不侵犯任何第三方的版权、专利、商标或其他权利；
3. 您有权签署本协议并授予上述许可。

### 5. 通知义务
如您提交的贡献包含第三方作品或需特别声明的内容，您同意在提交信息中显著标明。

### 6. 协议变更
项目管理方有权根据法律或项目发展需要修改本协议条款，修改后的条款将在官方渠道公布。

---
**签署方式：**
当您提交 Pull Request 或 Commit 到 e-dialect 相关仓库时，即视为您已阅读、理解并同意接受本协议条款的约束。
