# e-dialect 社区行为准则

## 我们的承诺

身为 e-dialect 社区的成员、贡献者和负责人，我们承诺：  

无论年龄、体型、身体状况、民族、性征、性别认同与表达、经验水平、教育程度、社会地位、国籍、相貌、种族、宗教信仰或性取向如何，  

每位参与本社区的人都应享有**免受骚扰**的环境。


我们致力于建设一个**开放、友善、多元、包容、健康**的社区，  

让方言保护与学习相关的讨论和协作，能够在安全、尊重的氛围中进行。

本行为准则适用于 e-dialect 组织下的所有仓库与社区空间。

---

## 我们的标准

### 有助于社区积极环境的行为（示例）

- 使用友好、包容的语言  
- 尊重不同的观点、经历和背景  
- 优雅地提出建设性批评，并耐心接受他人的批评  
- 在犯错时勇于承担责任，向受影响的人道歉，并从中学习  
- 关注社区共同利益，而非仅个人得失  
- 对其他社区成员展现同理心和善意

### 不当行为（示例）

- 使用与性有关的暗示性语言或图像，以及不受欢迎的性关注  
- 挑衅、侮辱、贬损性言论，或人身及政治攻击  
- 任何形式的公开或私下骚扰  
- 未经明确授权，擅自发布他人的私人信息，如地址、电子邮箱等  
- 其他在专业场景中可被合理认定为不当的行为

---

## 责任与权力

社区负责人有责任：

- 解释“何为妥当行为”，并依据本准则公平地认定和处理不当、威胁、冒犯或有害的行为；
- 有权利和义务删除、编辑或拒绝违反本准则的评论（comment）、提交（commit）、代码、Wiki 编辑、Issue 等贡献，并在必要时说明采取行动的理由。

所有社区成员都有义务配合维护一个健康的交流环境。

---

## 适用范围

本行为准则适用于：

- e-dialect 社区内的所有空间，包括 GitHub 仓库（Issue、Pull Request、代码、Wiki、讨论区等）、  
  官方社群（如 QQ 群、微信群）、官方邮箱与社交媒体账号等；
- 在其他场合代表 e-dialect 社区的情况，例如使用官方邮箱发言、作为指定代表参加线上或线下活动等。

## 举报与处理流程

### 如何举报

如遇到滥用、骚扰或其他不可接受的行为，请通过以下方式举报：

- 邮件：[edialect@edialect.top](http://mail.qq.com/cgi-bin/qm_share?t=qm_mailme&email=ocTFyMDNxMLV4cTFyMDNxMLVj9XO0Q)（社区公共邮箱）
- 或通过 GitHub 私信联系当前列出的社区负责人（在相关仓库 README 中注明）

所有投诉将由社区负责人或指定的行为准则工作组进行**及时、公正**的审查和调查。  
我们承诺对举报者的信息予以保密，并保护其不受报复。

### 处理分级

社区负责人将根据事件的影响和严重程度，采取相应措施：

1. **纠正（Correction）**  
   - 适用情形：使用不当语言或其他不符合职业操守或不受社区欢迎的行为。  
   - 处理方式：非公开书面警告，说明违规性质及为何不妥；可能要求公开道歉。

2. **警告（Warning）**  
   - 适用情形：单次或多次违规行为。  
   - 处理方式：发出正式警告，并在一定期限内限制违规者与当事人及社区负责人之间的联系，包括在社区空间及外部社交平台上的互动。若再犯，可升级为封禁。

3. **临时封禁（Temporary Ban）**  
   - 适用情形：严重违规或屡教不改。  
   - 处理方式：在规定时间内禁止参与社区的任何公开互动，包括 Issue、Pull Request、讨论群等。期间禁止任何形式的私下接触。

4. **永久封禁（Permanent Ban）**  
   - 适用情形：严重且持续的骚扰、攻击特定个人或群体等。  
   - 处理方式：无限期禁止该人员参与 e-dialect 社区的任何公开活动。

社区负责人保留在特殊情况下调整处理方式的权利，但会尽量保持公平、透明。

---

## 适用与变更

- e-dialect 社区负责人有权根据实际情况对本行为准则进行修订，重大变更会在社区渠道公示。
- 本行为准则自发布之日起生效。
